\doxysubsection{Подробное описание}
Класс для создания объекта конфигурации парсера. 

Класс \doxylink{class_parser_configuration}{Parser\+Configuration} предоставляет статический метод create() для создания объекта конфигурации парсера. Объект конфигурации может быть создан на основе файла конфигурации формата JSON. 

См. определение в файле \mbox{\hyperlink{_parser_configuration_8h_source}{Parser\+Configuration.\+h}} строка \mbox{\hyperlink{_parser_configuration_8h_source_l00016}{16}}

Виртуальная функция \doxylink{class_parser_configuration_a542aa87ab4a21ce9d52d388456ab8580}{get\+Integer()} возвращает целочисленное значение, соответствующее заданному ключу key.
