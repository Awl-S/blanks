\doxysubsection{Подробное описание}
Класс, представляющий интерфейс командной строки. 

Этот класс обрабатывает аргументы командной строки и категоризирует тип команды на основе аргументов. В настоящее время поддерживаются команды \textquotesingle{}convert\textquotesingle{}, \textquotesingle{}print\textquotesingle{} и \textquotesingle{}gen\textquotesingle{}.
