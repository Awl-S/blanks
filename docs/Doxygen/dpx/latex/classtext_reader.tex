\doxysubsection{Подробное описание}
Класс для чтения текстового файла. 

Класс \doxylink{classtext_reader}{text\+Reader} наследуется от класса \doxylink{class_file_reader}{File\+Reader} и предоставляет функциональность чтения различных типов данных из текстового файла.
