Структура для хранения данных типа zgt.

Структура zgt содержит поля для хранения различных данных типа zgt, которые представляют параметры для зубчатой передачи.

void zgt::display ( ) const
inline

Выводит содержимое структуры zgt на консоль.

Функция display() выводит значения каждого поля структуры zgt на консоль. Затем выводит значения отверстий для крепления, позицию маркера, серийный номер шрифта и позицию стопора.
