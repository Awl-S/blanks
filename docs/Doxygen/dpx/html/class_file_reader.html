Функция trim() удаляет ведущие и завершающие пробелы из строки s. Результат возвращается в виде новой строки.

Аргументы
sИсходная строка.
Возвращает
Строка без ведущих и завершающих пробелов.

Функция trim() удаляет ведущие и завершающие пробелы из строки s. Также удаляются любые символы, которые являются пробельными или имеют значение кода больше 255. Результат возвращается в виде новой строки.
