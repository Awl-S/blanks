Подробное описание

Класс для преобразования кодировки файлов.

Конструктор класса принимает входной и выходной каталоги. Если выходной каталог не указан, используется временный каталог.

Конвертирует файлы в указанной директории и её поддиректориях.

Метод рекурсивно конвертирует все файлы в указанной директории и её поддиректориях. Конвертируемые файлы должны иметь расширения, указанные в списке fileExtensions. Если файл имеет другое расширение и outputDirectory указана, он будет скопирован в указанную выходную директорию.

Аргументы
dirПуть к директории, в которой нужно выполнить конвертацию файлов.
